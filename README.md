
# Udacity Data Analyst Nanodegree: BikeShare Project

### Date created
2025-12-03

### Project Title
BikeShare Data Analysis Project

### Description
This project analyzes bike share data for three U.S. cities: Chicago, New York City, and Washington. 
The program allows the user to explore data, calculate statistics, and visualize insights about bike usage, trip duration, and user demographics.
